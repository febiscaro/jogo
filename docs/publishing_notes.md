# Publishing Notes

## Escopo atual

- Audio desabilitado por enquanto (intencional).
- Loop principal pronto: menu, tutorial, run roguelike, progressao e cosmeticos.
- Save local em:
  - `user://meta_progress.json` (meta da run)
  - `user://app_profile.json` (opcoes/cosmeticos/desbloqueios)

## Controles finais

- `A/D`: mover personagem no chao
- Mouse: apontar rolo livremente
- `Clique esquerdo` ou `P`: pintar
- `E`: recarregar no balde
- `1-4`: trocar cor
- `ESC`: pausar contrato

## Observacoes de UX

- Tutorial abre direto pelo menu.
- Contratos especiais so aparecem apos desbloqueio no perfil.
- Intensidade de pos-processamento e sensibilidade do cursor ficam em Opcoes.
